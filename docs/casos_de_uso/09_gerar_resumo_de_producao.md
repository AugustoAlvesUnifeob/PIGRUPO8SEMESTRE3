# Gerar Resumo de Produção

## Descrição
Gera um resumo automático da produção utilizando inteligência artificial.

---

## User Story

Como usuário da produção  
Quero receber um resumo automático da produção  
Para entender rapidamente o desempenho do dia.

---

## Caso de Uso

**Nome:** Gerar Resumo de Produção  
**Ator:** Usuário

### Fluxo Principal

1. O usuário acessa o dashboard.
2. O sistema coleta os dados de produção.
3. O sistema envia os dados para o módulo de IA.
4. A IA gera um resumo textual.
5. O sistema exibe o resumo no aplicativo.

---

## Critérios de Aceitação

- O sistema deve gerar um resumo automático.
- O resumo deve considerar dados de produção do dia.
