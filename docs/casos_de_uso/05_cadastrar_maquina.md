# Cadastrar Máquina

## Descrição
Permite cadastrar máquinas de corte no sistema para monitoramento da produção.

---

## User Story

Como usuário do sistema  
Quero cadastrar máquinas  
Para monitorar a produção de cada equipamento.

---

## Caso de Uso

**Nome:** Cadastrar Máquina  
**Ator:** Usuário

### Fluxo Principal

1. O usuário acessa a tela de cadastro de máquinas.
2. O sistema solicita os dados da máquina.
3. O usuário informa os dados.
4. O sistema registra a máquina.

---

## Critérios de Aceitação

- Deve permitir inserir identificação da máquina.
- Deve salvar os dados da máquina no sistema.
